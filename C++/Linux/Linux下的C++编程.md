### 1. Linux常用命令

三个最常用命令：`ls cd pwd`

1. ```
   ls -l -a -h -d	展示目录下文件和目录信息
   ```

2. ```
   cd		切换目录
   ```

3. ```
   pwd		当前目录下的绝对路径
   ```

4. ```
   mkdir	创建目录
   ```

5. ```
   rm	删除目录和文件
   ```

6. ```
   ll	等同于ls -l 目录下详细信息
   ```

7. ```
   mv   移动和更名
   ```

   

8. ```
   find 查找文件
   ```

9. ```
   grep 文本搜索
   ```

10. ```
    ping 测试连通性
    ```

11. ```
    netstat 
    ```
